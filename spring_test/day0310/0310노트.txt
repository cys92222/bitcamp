<< 자바스크립트의 for in >>
	==> 데이터를 갖고오는 것이 아니라, 인덱스를 갖고 와요!
	

var arr =["홍길동","이순신","유관순"]

for(var i in arr ){

	}
	
	===> 이때 i는 자바의 for( String i : list ){} 처럼 데이터를 갖고오는 것이 아니라 
		인덱스를 갖고와요. 
		i의 내용은 0,1,2 만큼 반복수행해요. 
		

String []arr = {"홍길동","유관순","이순신"};

for( String i : arr ){

	}
	
	=> 자바에서는 i에 데이터가 옵니다. 
		홍길동, 유관순, 이순신이 차례대로 i에 대입됩니다. 
		
		
for in을 객체에 사용할 수도 있어요.
즉, 객체의 속성 수 만큼 반복실행할때 사용해요. 

var person = {name:"홍길동", age:20, addr:"신수동", tel:"010-1234-5678"};
만약, person 객체의 속성 수 만큼 반복 실행해야 될 필요가 있다면 
for in을 사용해요.


for(var i in person){
		alert( person.i) 이렇게는 사용할수 없음주의!!! person객체에서 i 라는 속성을 찾아버린다.
		
		alert( person[i]); 속성명이 담긴 i를 변수처리해야하기 때문에 배열첨자 연산자를 사용해야함.
	}		
	
	이때 i는 person객체가 갖고있는 속성명이 차례대로 대입되어요.(name, age,addr,tel)

<< 자바스크립트에서 타이머 설정하기 >>

1. 지정한 시간이 되면 딱 한번 동작시키기
	==> setTimer( 함수, 시간 )


2. 지정한 시간마다 반복하여 동작시키기 
	==> setInterval( 함수, 시간 )
	
시간은 ms으로 준다. 1/1000s
	


function pro(){
		a;
		b;
	}


setTimer(pro, 2000) 
	==> 2초뒤에 pro함수를 한번만 실행하세요. 
	
setInterval(pro, 2000)	
	==> 2초마다 pro함수를 반복하여 실행하세요.
	
---------------------------------------------------------------------------
위에처럼하면 미리 기능을 만들어둬야하니까, 아래같은 방법을 더욱 많이 쓰인다.

setTimer( function(){
		a;
		b;
	} , 2000);


setInterval( function(){
		a;
		b;
	} ,2000 );

타이머를 설정할때 미리 함수를 만들어 둘수도 있고, 이름없는 함수를 매개변수로 바로 설정할 수도 있어요!
